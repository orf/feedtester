Feed creator

This is a very simple service that generates persistent updating RSS feeds.

Usage:

A feed can be accessed by using any string as a key by accessing this url: {% url "feeds:key" "KEY" %}. For example: {% url "feeds:key" uuid.hex %}

If you don't want to specify a key you can request {% url "feeds:cookie" %}. This will set a cookie which you will need to keep and re-send) with each request.

Options

By default a new item will be created when the feed is requested, if one hasn't been created in the last 30 seconds. You can pass an integer between 15 and 60 in the time parameter. To create a feed with a new entry every 20 seconds request this URL: {% url "feeds:key" uuid.hex %}?time=20
